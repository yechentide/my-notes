# 文字列

## 基本の使い方

### 要素の参照

```javascript
str[i]
str.at(i)
```

### 挿入・削除・置換

* 文字列の末尾への結合：`+=` `append()`
* 指定位置への挿入：`insert()`
* 指定範囲の削除：`erase()`
* 指定位置の置換：`replace()`
  コンストラクタと同様、様々な形式で引数を与えられる

### プロパティ

* length
    文字列の長さ
